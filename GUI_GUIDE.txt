TROUBLESHOOTING:
----------------
Problem: "LibreOffice Not Found" (red text at bottom)
Solution: Install LibreOffice from https://www.libreoffice.org/download/download/

Problem: GUI doesn't open
Solution: Make sure Python is installed. Run: python --version

Problem: Conversion fails
Solution: Check if the PPTX file opens in LibreOffice Impress
